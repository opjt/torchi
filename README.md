# torchi

> Push notification via HTTP API

`torchi`는 HTTP API 호출만으로 모바일에 즉시 푸시 알림을 보내고,  
알림에서 바로 **액션(수락 / 거절 / 응답)** 을 처리할 수 있는 가볍고 개발자 친화적인 푸시 알림 서비스입니다.

서비스 바로가기 -> [torchi.app](https://torchi.app)

**Key Features**

- PWA Based: 별도의 앱 설치 없이 웹에서 바로 시작하는 푸시 환경
- Developer Friendly: 간결한 API 구조와 쉬운 연동
- Interactive Actions: 알림에서 바로 실행하는 수락/거절/응답 액션
